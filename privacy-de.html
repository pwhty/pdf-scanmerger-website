2. Grundsatz

PDF ScanMerger wurde mit einem starken Fokus auf Datenschutz entwickelt. Die App läuft vollständig lokal auf Ihrem Mac. Es werden keine personenbezogenen Daten gesammelt, gespeichert, verarbeitet oder an Dritte übermittelt.

3. Datenerhebung

PDF ScanMerger erhebt keine Daten. Im Einzelnen:

  • Es werden keine Nutzungs- oder Analysedaten erhoben (kein Tracking).
  • Es werden keine Absturzberichte an den Entwickler gesendet.
  • Es werden keine personenbezogenen Daten (Name, E-Mail, Standort usw.) erhoben.
  • Es werden keine Dateien oder Dateiinhalte an externe Server übermittelt.
  • Die App initiiert von sich aus keine Netzwerkverbindungen.

4. Lokale Speicherung

Die App speichert ausschließlich Ihre Einstellungen (z. B. bevorzugte Ordnerpfade, Scan-Intervall) lokal auf Ihrem Mac mithilfe von macOS UserDefaults. Diese Einstellungen verlassen Ihren Computer niemals.

5. Dateizugriff

PDF ScanMerger verwendet die macOS-Sandbox und greift nur auf Dateien oder Ordner zu, die Sie ausdrücklich auswählen – per Drag & Drop, über einen Öffnen-Dialog oder einen konfigurierten Überwachungsordner. Der Zugriff wird über macOS Security-Scoped Bookmarks verwaltet.

6. Drittanbieterdienste

Die App verwendet keine Drittanbieter-SDKs, Werbenetzwerke, Analysetools oder Cloud-Dienste.

7. Kinder

Da die App keine Daten erhebt, gibt es keine besonderen Bestimmungen bezüglich Kindern. Die App ist für Nutzer aller Altersgruppen geeignet (Altersfreigabe 4+).
